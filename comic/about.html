How did I create the comic?

I created the comic with Microsoft Comic Chat. Comic Chat was a chat program, created by DJ Kurlander in Microsoft Research and released in 1996, that visually represented conversations in chat as sequences of comic panels. Users could chat on Internet Relay Chat (IRC) as comic characters, which was pretty groundbreaking in 1996.

Comic Chat uses AI (!) to automatically generate comic panels based on the text of the conversation. The characters can express different emotions based on the text. For example, if the user says, "I like that", the character will point at itself. If the user expresses anger, the character will frown or cross its arms.

A team in Microsoft Research called FUSE Labs worked on Comic Chat. FUSE Labs went on to create Bot Framework, which is part of the foundation of the Microsoft agent platforms today.

Comic Chat was discontinued in 2001 and the servers were shut down, but you can still use it on modern versions of Windows and connect to IRC servers.

Explore Pre-Built Agents First

Pre-built agents in Microsoft 365 Copilot are purpose-built AI assistants developed by Microsoft to enhance collaboration, productivity, and decision-making across the Microsoft 365 ecosystem. They are designed primarily for B2E (business-to-employee) scenarios. The pre-built agents work with organizational data through Microsoft Graph, respecting permissions and security controls.

Before diving into building an agent from scratch, take a moment to review the pre-built agents available in Microsoft 365 Copilot. Many common scenarios, like HR questions, meeting prep, writing assistance, and summarization, already have ready-to-use agents that may fully meet your needs.

Learn more about the pre-built agents in Microsoft 365.

Ask yourself:

  • Does one of the pre-built agents already cover most of my scenario?
  • Could I start with a pre-built agent and lightly customize it rather than building entirely new?
  • Does the way the pre-built agent handles permissions, knowledge, or actions fit my organization's requirements?

If the answer is yes, you can save time and effort by starting there.

If the answer is no, or if your use case requires deeper integration with business systems or more customization, then proceed to the platform questionnaire to identify the right path for your scenario.
